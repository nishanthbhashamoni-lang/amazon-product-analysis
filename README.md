# Amazon Product Analysis Dashboard

This project analyzes Amazon product data to identify trends in reviews, ratings, discounts, and category distribution.

## Tools Used
- Python
- Pandas
- Tableau

## Dashboard Features
- Total Reviews KPI
- Average Rating KPI
- Average Discount KPI
- Category Distribution
- Top 10 Most Reviewed Products
- Discount vs Rating Analysis

## Key Insights
• Products with higher discounts tend to attract more reviews  
• Electronics and accessories categories dominate the dataset  
• A small number of products generate the majority of reviews
